Projects

View your projects from your dashboard.

With the dashboard you can easily see a list of all of your existing projects. You can see information about the project such as the title, a description, the deadline and the time remaining.

Add a new project.

From this page you can also add a brand new project to your workstation. Doing so will generate a blank project for you to fill out as required.

Tasks

Simply track the status of tasks.

Using the "Tasks" page, simply keep track of a task with all the data needed. Wtihin the table you can see the other group members working on a specific task, add sub tasks to any task, see the start date and end date of any specific task. You can also see the status of the task at a glance with red being overdue, blue having time to be completed and green indicating a complete task.

Update a task.

From the tasks page you can also update any part of a task. You can assign additional members of the team to work a specific task, remove members from working the task, add a child task, mark the task as complete or delete the task and all of its child tasks.


Graphs

Gantt Charts

Generate different graphs to quickly assess the progress and current situation within the project. Using the Gantt chart you can see timescales involved within a project at a glance.

User Workload

See the breakdown of the tasks worked by different members of the group. Easily see which members are contributing the most work to the completion of the project.

Tree View

See which tasks lead on from other tasks and how an individual task relates to the rest of the project with an interactive tree layout of the tasks.
